### Reference Data Sources (Non-Canonical)

- `data/raw/nba_schedule.csv`
  - Source of NBA game listings and intended start times
- `data/raw/nba_team_map.json`
  - Source of team metadata keyed by `team_id`
- `data/external/nba_boxscores_team.json`
  - Team-level performance per game
- `data/external/nba_boxscores_player.json`
  - Player-level performance per game
